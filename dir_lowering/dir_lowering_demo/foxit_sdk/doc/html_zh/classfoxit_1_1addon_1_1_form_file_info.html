详细描述

此类表示表单文件信息。表单文件可以是 PDF、FDF、XML 或 XFDF 格式。

构造及析构函数说明

const WString &  password 
)

构造函数，带参数。

此构造函数将检查输入文件的有效性。

参数
[in]file_path表单文件的文件路径。这应该是 PDF、FDF、XML 或 XFDF 文件。
[in]password参数 file_path 指定的输入文件的密码。 这仅在输入文件是 PDF 文件时有用。如果输入文件不是 PDF 文件或 输入的 PDF 文件没有任何密码，请传递空字符串。

const WString &  password,
const wchar_t *  file_name 
)

构造函数，带参数。

此构造函数将检查输入文件的有效性。

参数
[in]reader_callback要与其他文件组合的文件的读取器文件回调。 它仅支持 PDF、FDF、XML 和 XFDF 文件。
[in]password参数 reader_callback 指定的输入文件的密码。 这仅在输入文件是 PDF 文件时有用。如果输入文件不是 PDF 文件或 输入的 PDF 文件没有任何密码，请传递空字符串。
[in]file_name参数 reader_callback 指定的输入文件的文件名。 当前信息对象在组合中使用时，此名称将被写入目标文件。 这可以是空字符串。
