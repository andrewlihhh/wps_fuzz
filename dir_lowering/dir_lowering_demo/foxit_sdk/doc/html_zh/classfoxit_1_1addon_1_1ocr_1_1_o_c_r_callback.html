详细描述

此类表示一个用于取消OCR进度的回调对象。 此类中的所有纯虚函数都用作回调函数，应由用户实现。 已实现的 OCRCallback 对象可以通过 函数 OCREngine::SetOCRCallback 设置。

◆ IsImageIgnored()

virtual bool foxit::addon::ocr::OCRCallback::IsImageIgnored ( foxit::pdf::graphics::ImageObject *  image_object)
pure virtual

用于确定是否应从OCR文本识别中排除图像对象的回调函数。

参数
[in]image_object指向 pdf::graphics::ImageObject 对象的指针。
返回
true 表示在OCR处理期间忽略该图像对象，该图像不会被OCR引擎处理用于文本提取，但图像本身仍保留在输出文档中， 而 false 表示不忽略。

◆ NeedToCancelNow()

virtual bool foxit::addon::ocr::OCRCallback::NeedToCancelNow ( const wchar_t *  info)
pure virtual

用于取消当前OCR进度的回调函数。

此函数不能用于取消OCR转换。

参数
[in]info描述OCR进度信息的字符串。
返回
true 表示取消进度，而 false 表示不取消。
