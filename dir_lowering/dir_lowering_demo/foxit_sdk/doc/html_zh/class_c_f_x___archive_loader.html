CFX_ArchiveLoader::CFX_ArchiveLoader ( FX_LPCBYTE  pData,

构造加载存档。

源数据缓冲区在反序列化期间必须保持有效。

参数
[in]pData要反序列化的源数据缓冲区。
[in]dwSize源数据缓冲区的字节长度。
