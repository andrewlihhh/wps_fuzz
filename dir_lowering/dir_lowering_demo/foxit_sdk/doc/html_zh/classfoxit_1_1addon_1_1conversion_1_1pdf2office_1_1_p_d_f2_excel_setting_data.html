enum foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::WorkbookSettings

Excel工作簿设置的枚举。

此枚举的值应单独使用。

枚举值
e_WorkbookSettingsDocument 

为文档创建单个工作表。

e_WorkbookSettingsEachTable 

为每个表格创建工作表。

e_WorkbookSettingsEachPage 

为每个页面创建工作表。

类成员变量说明

◆ decimal_symbol

String foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::decimal_symbol

一个字符串值，指示在PDF文档到Excel格式文档的转换过程中被识别为小数点的符号。

小数点符号用于分隔数字的整数部分和小数部分。 如果此参数设置为"."，则数字"123.456"将被识别为123.456。 如果此参数设置为","，则数字"123,456"将被识别为123.456。 如果此参数未设置，将自动识别小数点符号。

注解
目前，此参数仅对只包含数字和分隔符的单元格有用。 此参数的值必须与参数thousands_separator的值不同。 为了确保正确识别小数点符号，此参数应与参数thousands_separator一起设置。

◆ enable_aggressive_table_repair

bool foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::enable_aggressive_table_repair

一个布尔值，指示在将PDF文档转换为Excel格式文档时是否启用强化表格修复。默认值：true。

true表示在将PDF文档转换为Excel格式文档时启用强化表格修复。 false表示在将PDF文档转换为Excel格式文档时不启用强化表格修复。 启用后，该选项尝试保持表格的完整性并避免破坏表格结构。 它通过将重叠的元素移动到适当的边界，确保生成的Excel工作表中的单元格、行和列保持正确对齐。

◆ include_watermarks

bool foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::include_watermarks

一个布尔值，指示在将PDF文档转换为Excel格式文档时是否包含水印。默认值：false。

true表示在将PDF文档转换为Excel格式文档时包含水印。 false表示在将PDF文档转换为Excel格式文档时不包含水印。 启用该选项后，PDF文档中的水印将在转换过程中添加到Excel工作表中。 请注意，水印可能会与工作表内容重叠。某些单元格（例如页眉或区域列）可能会转换为文本框，在水印被移除或调整之前无法直接编辑。

◆ thousands_separator

String foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::thousands_separator

一个字符串值，指示在PDF文档到Excel格式文档的转换过程中被识别为千位分隔符的符号。

千位分隔符用于分隔数字的千位部分。 如果此参数设置为","，则数字"123,456"将被识别为123456。 如果此参数设置为"."，则数字"123.456"将被识别为123456。 如果此参数设置为" "，则数字"123 456"将被识别为123456。 如果此参数未设置，将自动识别千位分隔符。

注解
目前，此参数仅对只包含数字和分隔符的单元格有用。 此参数的值必须与参数decimal_symbol的值不同。 为了确保正确识别千位分隔符，此参数应与参数decimal_symbol一起设置。

◆ workbook_settings

WorkbookSettings foxit::addon::conversion::pdf2office::PDF2ExcelSettingData::workbook_settings

一个枚举值，指示PDF文档到Excel格式文档转换的工作簿设置。

请参考从PDF2ExcelSettingData::e_WorkbookSettingsDocument开始的值，这应该是这些值中的一个。 默认值：PDF2ExcelSettingData::e_WorkbookSettingsEachPage。
