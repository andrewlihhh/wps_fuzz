详细描述

template<class ObjectClass>
class CFX_ObjectArray< ObjectClass >

表示对象数组模板的类。 数组中的每个对象都被数组"拥有"，因此当此数组被析构时， 或从数组中移除对象时，对象将被析构。

模板参数 <class T> 必须支持"placement new"。 （任何从CFX_Object派生的类都支持placement new）。
