详细描述

此类表示用于将PDF文档转换为PowerPoint格式文档的设置数据。

带参数的构造函数。

参数
[in]enable_aggressively_split_sections一个布尔值，指示是否在PDF文档到PowerPoint格式文档的转换中激进地分割节。默认值：false。 在同一文本框的两个段落中，如果它们之间的间距足够大，它们将被分割为两个独立的文本框 在PDF文档到PowerPoint格式文档的转换中。 true表示在PDF文档到PowerPoint格式文档的转换中激进地分割节。 false表示在PDF文档到PowerPoint格式文档的转换中不激进地分割节。
[in]enable_adapt_to_largest_page一个布尔值，用于指示在将 PDF 文档转换为 PowerPoint 格式时，是否将较小页面的内容适配到最大页面尺寸。 默认值为 false。 由于 PowerPoint 中的所有幻灯片必须具有相同的尺寸，默认情况下会以最大页面作为参考尺寸。 启用该选项后，转换器会在转换范围内识别最大的页面，并相应地调整较小页面的页面矩阵，对其图形对象进行缩放，以确保输出的 PowerPoint 文件中所有幻灯片尺寸一致。 所有缩放操作都会保持原始宽高比不变。 true 表示将较小页面的内容适配到最大页面尺寸。 false 表示不将较小页面的内容适配到最大页面尺寸。

bool foxit::addon::conversion::pdf2office::PDF2PowerPointSettingData::enable_adapt_to_largest_page

一个布尔值，用于指示在将 PDF 文档转换为 PowerPoint 格式时，是否将较小页面的内容适配到最大页面尺寸。 默认值为 false。

由于 PowerPoint 中的所有幻灯片必须具有相同的尺寸，默认情况下会以最大页面作为参考尺寸。 启用该选项后，转换器会在转换范围内识别最大的页面，并相应地调整较小页面的页面矩阵，对其图形对象进行缩放，以确保输出的 PowerPoint 文件中所有幻灯片尺寸一致。 所有缩放操作都会保持原始宽高比不变。 true 表示将较小页面的内容适配到最大页面尺寸。 false 表示不将较小页面的内容适配到最大页面尺寸。

bool foxit::addon::conversion::pdf2office::PDF2PowerPointSettingData::enable_aggressively_split_sections

一个布尔值，指示是否在PDF文档到PowerPoint格式文档的转换中激进地分割节。默认值：false。

在同一文本框的两个段落中，如果它们之间的间距足够大，它们将被分割为两个独立的文本框 在PDF文档到PowerPoint格式文档的转换中。 true表示在PDF文档到PowerPoint格式文档的转换中激进地分割节。 false表示在PDF文档到PowerPoint格式文档的转换中不激进地分割节。
