详细描述

此类表示对话框元素属性，指定具有以下属性集的对象字面量。

int foxit::DialogDescriptionElement::char_height

指定元素的高度（以字符为单位）。

◆ char_width

int foxit::DialogDescriptionElement::char_width

指定元素的宽度（以字符为单位）。

如果未指定宽度，则使用内容的组合宽度。

◆ font

String foxit::DialogDescriptionElement::font

此元素使用的字体。

它将是以下字符串之一：

  • "default" - 默认字体
  • "dialog" - 对话框字体
  • "palette" - 调色板（小）字体
