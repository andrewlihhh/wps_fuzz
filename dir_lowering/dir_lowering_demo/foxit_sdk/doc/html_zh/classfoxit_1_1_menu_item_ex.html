类成员变量说明

◆ is_checked

bool foxit::MenuItemEx::is_checked

布尔值，指示菜单项是否标记为选中状态。

true 表示菜单项将被标记为选中状态， false 表示菜单项不会被标记为选中状态。

◆ is_enabled

bool foxit::MenuItemEx::is_enabled

布尔值，指示菜单项是否显示为启用状态或灰显状态。

true 表示菜单项将显示为启用状态， false 表示菜单项将显示为灰显状态。
