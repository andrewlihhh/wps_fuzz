详细描述

此类表示拆分生成的PDF文档的结果信息。

成员枚举类型说明

◆ SplitFileState

enum foxit::pdf::SplitFileResult::SplitFileState

拆分文件状态的枚举。

此枚举的值应该单独使用。

枚举值
e_SplitFileSuccess 

状态为拆分成功。

e_SplitFileErrSizeTooSmall 

状态为因文件大小过小而失败。

@detail 如果要拆分的PDF文档的文件大小小于StartSplitByFileSize函数中参数 max_file_size 的值， 则此状态将在文件拆分结果中返回。

e_SplitFileErrFileExist 

状态为因文件存在而失败。

@detail 如果拆分产生的PDF文档已经存在，且SplitOutputSettingData对象中的 is_overwrite 值 设置为false，则此状态将在文件拆分结果中返回。

e_SplitFileErrAccessDeny 

状态为因文件访问被拒绝而失败。

@detail 如果要拆分的PDF文档被拒绝访问，则此状态将在文件拆分结果中返回。状态为因文件访问被拒绝而失败。
