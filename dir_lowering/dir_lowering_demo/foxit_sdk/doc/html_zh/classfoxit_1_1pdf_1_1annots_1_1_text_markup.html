详细描述

文本标记注释在文档文本中显示为高亮、下划线、删除线或波浪线。 文本标记注释都是标记注释，因此类 TextMarkup 派生自 类 Markup。它是所有PDF文本标记注释的基类。它提供 获取/设置文本标记注释通用属性的基本函数。
特别地，对于文本标记注释，四边形是必需的。因此请确保文本标记 注释在重置其外观流之前具有有效的四边形。
有关具体的文本标记注释类型，请参考派生类。

◆ GetQuadPoints()

QuadPointsArray foxit::pdf::annots::TextMarkup::GetQuadPoints ( ) const

获取四边形。

此属性影响文本标记注释的外观。四边形中点的顺序应为：

  • 第一个点是四边形左上角的点。
  • 第二个点是四边形右上角的点。
  • 第三个点是四边形左下角的点。
  • 第四个点是四边形右下角的点。
返回
接收四边形值的四边形点数组。

◆ SetQuadPoints()

void foxit::pdf::annots::TextMarkup::SetQuadPoints ( const QuadPointsArray &  quad_points_array)

设置四边形。

对于文本标记注释，四边形是必需的。此属性影响文本标记 注释的外观。设置新四边形后，当前文本标记注释的矩形也会被更新。 四边形中点的顺序应为：

  • 第一个点是四边形左上角的点。
  • 第二个点是四边形右上角的点。
  • 第三个点是四边形左下角的点。
  • 第四个点是四边形右下角的点。

如果用户希望新四边形对注释外观有影响，请在设置四边形后调用函数 Annot::ResetAppearanceStream。

参数
[in]quad_points_array有效的四边形数组。不应为空数组。
返回
无。
